# Cash Matrix Analysis

## Overview
This project analyzes fintech business account data inspired by Cash Mtrix, covering account Opening records, inward and outward transaction, bills payment. the purpose is to understand business behavior, cash flow performance and financial integrity withing fintech environment.
An interactive power bi dashboard was created to ucover insights that can guide the app.

Cash matrix is one of the fintech company providing mobile payment, digital wallets, marchant services, and bill payment solution.
